# 🚀 GoSmart - Smart Ride, Fair Price

**GoSmart** ایک جدید رائیڈ ہیلنگ ایپ ہے جو Next.js پر بنائی گئی ہے۔

### 🌟 اہم فیچرز:
- **Electric Indigo UI:** جدید اور خوبصورت ڈیزائن۔
- **Smart Fare:** خودکار اور منصفانہ کرایہ۔
- **5% Flat Commission:** ڈرائیورز کے لیے سب سے زیادہ بچت۔
- **All-in-One Dashboard:** رائیڈر، ڈرائیور اور ایڈمن ایک ہی جگہ۔

### 🛠 ٹیکنالوجی:
- Next.js, Tailwind CSS, Lucide-React.

### 🚀 ڈیپلائمنٹ:
ورسل (Vercel) پر ایک کلک سے لائیو کریں۔
